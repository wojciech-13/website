About this site

This is example site which shows a few things made with CSS, JAVASCRIPT and HTML .

You can find here four subpages: Cube, Clock, Painting, Game

Cube

This is a rotating cube created with CSS animations. The buttons above her allows you to change the state of animation - start, pause, resume or reset. Try it yourself.

Clock

Just a graphic clock made by CSS3 and some Javascript. It was used data object to import the current time and setInterval function to rotate hands of the clock every second.

Painting

On this subpage you can draw your own painting. You have two range buttons that you can use for changing line color or line width. It was made thanks to canvas from HTML5.

Use clear button if you want to draw again.

Game

The last one is the game, created with canvas in Javascript. New drawing is created every 5 miliseconds and replaces the old one. It gives the impression that the ball is moving. Use your mouse to control paddle and try win.

- There is 40% to bounce ball symetrically to paddle.
- The first one who score 5 points will win.
- GOOD LUCK!
